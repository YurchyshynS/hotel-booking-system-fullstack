# Test Plan

## Project Name
Hotel Booking System

## Objective
Verify functionality, stability and business logic of hotel booking web application.

## Scope

### In Scope
- Registration / Login
- Room listing
- Booking creation
- Booking cancellation
- Profile page
- Validation rules
- UI behavior

### Out of Scope
- Performance testing
- Security penetration testing
- Mobile native app

## Test Types
- Smoke Testing
- Functional Testing
- Negative Testing
- UI Testing
- Regression Testing

## Environment
- Windows 11
- Chrome
- Localhost environment

## Entry Criteria
- Application starts successfully
- Database connected

## Exit Criteria
- Critical bugs fixed
- Smoke suite passed
